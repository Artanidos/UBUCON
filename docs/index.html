Do you know what UBUNTU is all about?
UBUNTU means, I am, because we are. We are helping each other, without asking for money or something in exchange.
As Meli asked me in August 2020, if I would start this project for the UBUNTU movement, then I immediatly said, YES.
I was just happy to get this project. All my life I was steering into this direction. I felt the call years before. It is my biggest project so far, even after being a software developer since 30 years now.
I am not good in anything in particular, but my skills are cross plattform, user interface design and user centered design that is what is needed right now.

Maybe later we need to crowdfund some budget, because we might need hardware or we might have to pay some developer, but right now we just need time.

Ideas

Here are some ideas what we can implement with UBUCON as plugin.

Housing

There are many houses abandoned on the whole planet while we have got many homeless people. What if we track these house with geo locations, pictures and status, so that everyone can go there an occupy it.
These houses will get a traffic light system. Red means already occupied. Green means empty. And orange means somebody is living there an wants someone else to join.

Travel

You want to travel. You have got a spare room. These this plugin can help you to connect each other. We don't have to pay for expensive hotels.

Ride Share

You want to travel with your car and have got spare space, then this plugin may connect people.

Hands

You need help in the garden for harvesting food. This plugin might connect you with helpers.
